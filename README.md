# Pragmatic nested list

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/~/github.com/manu180/pragmatic-nested-list-example)

An example on how to handle drag'n'drop with *nested lists* using [Pragmatic drag and drop](https://github.com/atlassian/pragmatic-drag-and-drop) and leverages the [list-item hitbox](https://atlassian.design/components/pragmatic-drag-and-drop/optional-packages/hitbox/about#list-item).

This project is powered by [Vite](https://vitejs.dev/).

It is mainly based on the following technologies:
- [TypeScript](https://www.typescriptlang.org/)
- [React](https://react.dev/)
- [TailwindCSS](https://tailwindcss.com/)
- [Lucide](https://lucide.dev/)


## Running locally

```bash
npm install
npm run dev
```
